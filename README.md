# vjemmie
Discord bot powered by [discord.py](https://github.com/Rapptz/discord.py)

Requirements
------------
* Python >=3.7
* FFMPEG on PATH


Installing
----------

Download or clone git repo

```sh
git clone https://github.com/PederHA/vjemmie.git
cd vjemmie
pipenv install
```

Rename `secrets_example.py` to `secrets.py` and add credentials.
